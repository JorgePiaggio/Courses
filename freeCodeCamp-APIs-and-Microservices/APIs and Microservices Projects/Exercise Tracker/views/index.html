Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

Get User Id

POST /api/exercise/userid

GET users's id
GET /api/exercise/user-id

GET users's exercise log
GET /api/exercise/log?{userId}[&from][&to][&limit]

EXAMPLE GET users's exercise log
https://fcc-exerc-trakrr.glitch.me/api/exercise/log?userId=60d2274275c3cc6270350

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
