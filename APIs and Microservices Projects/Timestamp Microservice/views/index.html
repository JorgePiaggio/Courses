Timestamp Microservice

Example Usage:


  • Date Format Request
    [project url]/api/timestamp/2021-01-03

  • Unix Timestamp Request
    [project url]/api/timestamp/1609632000000

  • Invalid Request
    [project url]/api/timestamp/gdsaggs

Example Output:

{"unix":1609632000000, "utc":"Sun, 03 Jan 2021 00:00:00 GMT"}
